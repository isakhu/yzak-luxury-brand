# Product images

Place image files in the matching category folder:

- `gold/` — gold jewelry
- `diamonds/` — diamond jewelry
- `clothes/` — clothing
- `shoes/` — shoes
- `watches/` — watches
- `handbags/` — handbags
- `accessories/` — accessories

**Example:** For a gold ring, save `gold-ring.jpg` as:

`public/images/gold/gold-ring.jpg`

In the admin panel, enter the filename only: `gold-ring.jpg`

The site displays it at: `/images/gold/gold-ring.jpg`
