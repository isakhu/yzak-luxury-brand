# YZAK Luxury Brand

A premium digital brand experience — high-end UI/UX design system built for luxury aesthetics.

## Live Demo

🔗 [yzak-luxury-brand.vercel.app](https://yzak-luxury-brand.vercel.app)

## About

This project is a digital brand identity system for a luxury fashion label. The focus is on visual storytelling, refined typography, and smooth interactions that match the premium feel of the brand.

## Features

- Premium dark aesthetic with gold accents
- Smooth page transitions and micro-interactions
- Fully responsive across desktop, tablet, and mobile
- Optimized for performance and Core Web Vitals

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS |
| Language | TypeScript |
| Deployment | Vercel |

## Getting Started

```bash
git clone https://github.com/isakhu/yzak-luxury-brand
cd yzak-luxury-brand
npm install
npm run dev
```
